実体配線図ベースで初心者からプロまで使えるシミュレーター
---
## 特徴
- 実態配線図ベース
  - カラーコードなども自動変更
## 使い方
### 1.  [BitGate](https://f69-js.github.io/BitGate/)にアクセス
### 2.  部品を配置する
左側にある部品のボタンを押して部品を配置
### 3. 値の変更
部品をクリックして上にある値を変更する
### 4. シミュレーション
上にある`RUN SYSTEM`(緑)ボタンを押す
オレンジ色になったら成功
### 5 .部品の移動と削除
移動: 配線や部品をクリックして移動
削除: 配線や部品をクリックして`DELETE`ボタンを押す

## QA
- Q:ズームイン、ズームアウトはどうしますか
  - A:マウスのホイールです
- Q:グリッドの移動はどうしますか
  - A:スペースを押しながらドラッグします
- Q:NOTが動作しません。
  - A:VCCとGND(色がついているピン)に電気が来ていて、緑色のテキストが出ていますか？
